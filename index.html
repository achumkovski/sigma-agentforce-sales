01Autonomous SDR / Prospecting agent
Autonomous
 Handles outbound sequencing and qualifies prospects through multi-turn conversation, escalating to a human rep only when ready for discovery. Outbound is inconsistent and rep-dependent — pipeline suffers whenever reps are too busy to prospect. "I want to maintain a consistent outbound motion across all target accounts so that pipeline never dries up between campaigns."
SDR managersVP SalesRev Ops
 Always-on pipeline generation without headcount dependency or cadence gaps.

The visit cycle
06Post-visit debrief agent
 A voice memo from the car park triggers the agent to update the opportunity, create and assign action items, draft follow-up emails, send calendar invites, and log the visit — all parsed from natural language. End-of-day logging is time-consuming and inaccurate; action items slip and pipeline data degrades. "I want to capture everything from my meetings in under two minutes so that my pipeline data is always current and follow-ups never fall through."
Field reps
 Real-time data accuracy across the pipeline — the foundational input that powers every downstream agent.
  1. When do reps typically update Salesforce after a customer visit — same day or later?
  2. What percentage of field visits result in a complete opportunity update?
  3. Do reps currently use the Salesforce mobile app during or after visits?

"I want my pipeline continuously reviewed in the background so that I always know which deals need attention before they slip the quarter."

  3. Do you apply a formal sales methodology (e.g. MEDDIC), and how consistently is it used?

"I want a fully drafted QBR ready for my review so that I walk into every customer review focused on the conversation, not the slides."

 Parses incoming RFP requirements, maps them against product capabilities and past win content from the library, and generates a first-draft response for the rep to review and approve — compressing proposal cycles from days to hours. "I want a first-draft proposal generated from requirements in minutes so that I can respond to more opportunities without compromising on quality."
